Add Product → Webhooks

Đây là cách app nhận event check-in/comment real-time.

  • Topic: Page
  • Callback URL: https://api.dhtcdanang.com/api/v1/meta/webhook
  • Verify Token: random string lưu trong backend/.env với key META_WEBHOOK_VERIFY_TOKEN
  • Subscribe fields (3 cái cho check-in reply flow):
    • feed — bắt mọi post/comment tag Page (gồm check-in)
    • messages — bắt DM từ khách
    • messaging_postbacks — bắt click button trong Messenger

Add Product → Messenger → Settings

  • Generate Page Access Token cho Page DHTC (long-lived token)
  • Save token vào backend/.env key META_PAGE_ACCESS_TOKEN
  • Subscribe Page vào webhook fields: messages, messaging_postbacks
Bảo mật: Page Access Token cấp quyền gửi DM dưới danh nghĩa Page. KHÔNG commit lên git, KHÔNG log ra stdout. Chỉ lưu trong .env.
